# otakular
website for classification of brain tumor MRI images using the python programming language and the KNN algorithm<br>
Here modules :
<ol>
  <li>Upload MRI</li>
  <liData Klasifikasi / Classification Data</li>
  <li>Data Akurasi / Accuracy Data</li>
  <li>Set K</li>
</ol>
